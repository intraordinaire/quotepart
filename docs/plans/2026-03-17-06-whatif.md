# Plan 06 — "Et si..." Simulator

> **For agentic workers:** REQUIRED: Use `superpowers:executing-plans` to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the "Et si..." free-form scenario editor — pre-filled with current simulation values, any field editable, results recalculate in real-time, split-screen before/after view with highlighted deltas.

**Architecture:** `WhatIfContext` wraps a local copy of `SimulationInput` initialized from the main `SimulationContext`. Edits are local — they never mutate the "real" simulation. `calculate()` runs on every state change (debounced). Split screen shows frozen "before" snapshot on the left, live "after" on the right. "Copy link" encodes the modified state into a new URL.

**Tech Stack:** React, TypeScript, Tailwind. `useDeferredValue` or debounce for real-time calc.

**Prerequisite:** Plans 01 + 02 + 03 + 04 complete.
