# Plan 04 — Results Screen

> **For agentic workers:** REQUIRED: Use `superpowers:executing-plans` to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the results screen: comparison table (5 models), equity gauges, temporal projections, per-model detail view, and (in couple mode) domestic perception confrontation.

**Architecture:** Results are derived from `SimulationContext` state → passed through `calculate()` → rendered as pure display components. No local state except UI toggles (expanded detail view). Locked/grayed models for unfinished tiers. Click-to-detail panel per model.

**Tech Stack:** Next.js, React, TypeScript, Tailwind. No new dependencies.

**Prerequisite:** Plans 01 + 02 + 03 complete.
